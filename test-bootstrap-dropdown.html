Test Bootstrap Dropdown Nativo

Esta página prueba el dropdown usando Bootstrap nativo sin JavaScript personalizado.

Dropdown de Reportes (Bootstrap Nativo):

  • Reporte Diario
  • Reporte Semanal
  • Reporte Mensual
  • Reporte Anual
  • Reporte por Fechas
Estado: Bootstrap inicializado correctamente
Instrucciones de Prueba:
  1. Haz clic en el botón "Reportes" para abrir el dropdown
  2. Haz clic fuera del dropdown para cerrarlo
  3. Presiona la tecla Escape para cerrarlo
  4. Selecciona una opción del dropdown
Características de Bootstrap:
  • ✅ Z-index automático
  • ✅ Posicionamiento correcto
  • ✅ Cierre automático al hacer clic fuera
  • ✅ Soporte para tecla Escape
  • ✅ Responsive
